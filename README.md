# StockYogi
A Full-stack AI-powered project to provide a one-stop destination for all things stock market.
From live market updates, global trends to real-time stock prices and AI-assistance.
This project has the following features:
1. Real-time Market News
2. Real-time index/stock prices
3. An innovative mutual fund screener that suggests mutual fund porftolio with distribution based on user risk apetite.
4. AI-Assistance for market sentiments and possible trends
   
# Tech Stack

**Frontend**: ReactJS

**Backend** : ExpressJS

**Runtime Environment** : NodeJS

**Database** : MongoDB

# Project Architecture

**Frontend**: A React-App for all the frontend routes of the application

**Backend** : The main backend app containing all the routes of the application. Implemented using NodeJs.  

**Python Backend** : A python backend to be used for fetching stock data using yfinance.
